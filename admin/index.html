✏️ Editor

Guarda antes la imagen en la carpeta assets/img/. Aquí escribe solo el nombre del archivo. Si lo dejas vacío, se pone un gráfico de marca automático.

Formato sencillo: ## Subtítulo · - elemento de lista · ![texto](imagen.jpg) para una imagen · línea en blanco = nuevo párrafo.

¿Y ahora qué hago con el archivo?
  1. Si usas imágenes, súbelas primero a la carpeta assets/img/.
  2. Se descarga un archivo .html con tu artículo ya maquetado.
  3. Súbelo a la carpeta articulos/ de tu blog (o pásamelo y lo publico yo).
  4. ¡Listo! Ya está online en tu dominio.
